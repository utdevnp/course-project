
## Course Project 

### How to start
<b>Download Project  </b>  
<pre>
    git clone https://github.com/utdevnp/course-project.git
</pre>
<b>Install all depandency from npm </b>  
<pre>
    npm install
</pre>
<b> Start application </b> 
<pre>
    npm start
</pre>
<b>Start test</b>
<pre>
    npm test
</pre>
